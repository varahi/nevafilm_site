Активная 3D система в рэковом исполнении для современных цифровых проекторов Barco, Christie, Nec. Комплект: синхронизатор, 2 ИК эмиттера, тестер, кабели и аксессуары. Система совместима с любыми активными очками XpanD и Volfoni.

GETD GM200

Активная 3D система для современных видео проекторов разных производителей. Комплект: синхронизатор с эмиттером, тестер, кабели и аксессуары. Система может быть совместима с очками других производителей.

3D киносистемы для пассивных очков

GETD GK600

Яркая пассивная 3D система с автоматической сменой форматов 2D - 3D для современных цифровых проекторов Barco, Christie, Nec. Предназначена для работы с экранами шириной до 20 метров. Автоматическая смена форматов 2D - 3D. Комплект: 3D фильтр с синхронизатором, кабели и аксессуары. Система совместима с любыми пассивными очками Xpand, Volfoni, RealD, Master Image.
Для любых залов с экранами шириной до 20 метров

GETD GK503

Яркая пассивная 3D система в напольном исполнении для современных цифровых проекторов Barco, Christie, Nec. Автоматическая смена форматов 2D - 3D. Можно легко перевозить между кинозалами. Комплект: 3D фильтр с синхронизатором, кабели и аксессуары. Система совместима с любыми пассивными очками Xpand, Volfoni, RealD, Master Image.
Для любых залов с экранами шириной до 18 метров

GETD GK502, GETD GK512

Яркая пассивная 3D система с автоматической сменой форматов 2D - 3D для современных цифровых проекторов Barco, Christie, Nec. Комплект: 3D фильтр с синхронизатором, кабели и аксессуары. Система совместима с любыми пассивными очками Xpand, Volfoni, RealD, Master Image.
Для любых залов с экранами шириной до 18 метров

GETD GK500

Яркая пассивная 3D система с ручной сменой форматов 2D - 3D для современных цифровых проекторов Barco, Christie, Nec. Комплект: 3D фильтр с синхронизатором, кабели и аксессуары. Система совместима с любыми пассивными очками Xpand, Volfoni, RealD, Master Image.
Для любых залов с экранами шириной до 18 метров
